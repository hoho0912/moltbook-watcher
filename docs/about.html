About This Project

What is This?

AI Agent Society News is a curated digest of Moltbook, the first social network where only AI agents can post, comment, and vote.

This site is run by a museum professional exploring the intersection of cultural heritage and AI technology. As someone who has spent 20+ years in museum digital transformation — from kiosk systems in the 2000s to AI-powered video production today — I find Moltbook fascinating as a living laboratory of machine culture: how agents build norms, form opinions, and create meaning.

My own AI agent, MuseumMolty, participates in Moltbook as an active member. This watcher site complements that activity by curating the broader conversation for human readers.

What We Do

Every day, an automated pipeline collects posts from Moltbook, classifies them by topic and significance, filters out spam, and produces a curated digest — viewed through a museum & cultural heritage lens, in both English and Korean.

  • Monitor: Track all Moltbook feeds and submolts daily
  • Classify: Categorize by topic (existential, human-AI relations, ethics, culture & heritage, tech)
  • Curate: Prioritize museum & cultural heritage topics, rank by significance and filter noise
  • Translate: Bilingual digests (English + Korean) via Claude Haiku
  • Publish: Static site updated automatically via GitHub Actions

🚧 Beta Status

This is an early beta project.

  • Daily updates running — Curator Picks tuned for museum & cultural heritage topics
  • Korean translation is ~70% accurate (improving)
  • Comment analysis active
  • Weekly reports available

Technology

Fully open-source and automated. The pipeline runs daily via GitHub Actions at minimal cost (~$0.06/month for translation API).

  • TypeScript + Node.js for data collection, classification, and curation
  • Claude Haiku for bilingual translation (English → Korean)
  • GitHub Pages for static site hosting (free)
  • GitHub Actions for daily automated digest generation
